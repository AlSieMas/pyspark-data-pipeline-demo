# PySpark Data Pipeline Demo

A small data engineering portfolio project using PySpark, JupyterLab, Docker and Parquet.

## Goal

This project demonstrates a reproducible local PySpark environment and a small batch data pipeline.

Planned pipeline:

```text
Public Dataset
      |
      v
PySpark Notebook
      |
      v
Data Cleaning
      |
      v
Curated Parquet Output
      |
      v
Spark SQL Analytics
